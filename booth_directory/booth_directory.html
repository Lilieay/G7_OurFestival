ร้าน EduPath

EduPath ร้านหนังสือสำหรับกลุ่มคนที่จะกล่าวเข้าสู่มหาวิทยาลัย รวมรวบคลังความรู้สำหรับการเข้ามหาวิทลัยไว้มากมายพร้อมทริคติวสอบเข้ามหาแบบเข้าใจง่ายสุดๆ เพราะเราเชื่อว่าเส้นทางเข้ามหาลัยที่ไม่ต้องเดินคนเดียวนั้นสดใสกว่า ❤️ มาก้าวไปด้วยกันกับพวกเรา EduPath!

ร้าน KanitBook

KanitBook ร้านหนังสือแห่งเวทมนตร์การเรียนรู้ ที่รวบรวมหนังสือคณิตศาสตร์และความรู้ทั่วไปไว้ให้คุณได้เลือกสรร เพื่อเสริมสร้างความเข้าใจและแรงบันดาลใจในการศึกษา ในความมืดของคืนฮาโลวีน ยังมีแสงไฟจากหน้าหนังสือที่รอให้คุณเปิดอ่าน ทุกเล่มคือเวทมนตร์แห่งความรู้ ที่จะปลุกพลังแห่งการเรียนรู้ในตัวคุณ

ร้าน The Fictional Alibi

The Fictional Alibi คือ "แหล่งซ่อนตัว" ที่สมบูรณ์แบบ เราได้คัดสรรนิยายและวรรณกรรมชั้นดีจากทั้งในและต่างประเทศ ที่จะพาคุณ "หลบหนี" จากความจริงอันวุ่นวาย เราขอเตือนไว้ก่อน... เมื่อคุณอ้าง "ที่อยู่" (Alibi) ของคุณกับเรา คุณอาจไม่อยาก "กลับ" ไปสู่โลกเดิมอีกเลย
